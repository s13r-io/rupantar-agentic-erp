# Rupantar Horizon 1 Prototype Documents

This folder contains four shareable Markdown documents:

1. **01_horizon1_prototype_scope.md**  
   Horizon 1 prototype scope, goals, target audiences, what should and should not be in v1, success criteria, and recommended packaging.

2. **02_prototype_blueprint.md**  
   Prototype blueprint with screens, user roles, demo scenarios, feature scope for v1, real vs simulated boundaries, and 5-minute / 15-minute walkthroughs.

3. **03_wireframe_spec.md**  
   Screen-by-screen wireframe specification with exact UI content, field definitions, screen map, scenario copy, and Figma frame organization.

4. **04_component_inventory_and_interaction_spec.md**  
   Reusable component inventory, card types, stepper states, button hierarchy, message states, audit event schema, and seeded data JSON.

These documents are intended to be shared directly with:
- product
- design
- engineering
- leadership
- investor / partner demo prep teams

All four are written in Markdown for easy editing, versioning, and sharing.
