## What this repo is

**Rupantar AI** — strategic documentation + prototype planning for an AI-powered agentic ERP for India's 50M SMB shops. No application code exists yet. Two active work areas:

- `docs/` — Chairman-facing proposal deliverables (Executive Brief, Full Proposal, Appendices)
- `planning/` — Horizon 1 prototype scope, wireframes, component specs, and developer handoff

## What Rupantar AI is (never get this wrong)

One system: the **Agentic Operations Platform** — 160+ pre-built AI agents for shop ops (billing, inventory, credit, compliance), composed by partners into bespoke solutions per shop, delivered via WhatsApp and voice. Distribution is through certified "AI Shop Doctors" (India's existing field force). Do NOT say "product + platform" or "two things we build."

## Language rules for document work

- Use: "platform," "foundation," "digital layer," "ecosystem"
- Avoid in thesis content: "infrastructure," "DPI," "backbone," "monopoly"
- Audience for `docs/`: the chairman — key insights first, no fluff
- All market data must be sourced (SaaSBoomi, KenResearch, Mordor Intelligence, NPCI, MSME Ministry). Flag estimates.
