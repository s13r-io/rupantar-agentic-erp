- Key insight goes in the first sentence of each section — never buried
- Part 2 of the Full Proposal: opportunity before solution (§2.1–§2.4 order must be preserved)
- Use "Agentic Operations Platform" not "product + platform"
- Use "Ecosystem Readiness Scorecard" not "Platform Readiness Scorecard"
